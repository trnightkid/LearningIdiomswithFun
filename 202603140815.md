```python
import hashlib

class BlacklistMatcher:
    def __init__(self, salt, hash_list):
        """
        初始化匹配器
        :param salt: 盐值（我提供）
        :param hash_list: 哈希库（我提供）
        """
        self.salt = salt
        self.hash_set = set(hash_list)  # 转为集合加速查找
    
    def hash_domain(self, domain):
        """
        对域名进行哈希（与我的生成方式一致）
        """
        # 1. 统一小写
        domain = domain.lower().strip()
        
        # 2. 加上盐值
        combined = self.salt + domain
        
        # 3. SHA-256 哈希
        return hashlib.sha256(combined.encode('utf-8')).hexdigest()
    
    def check(self, domain):
        """
        检查域名是否在黑名单中
        :param domain: 用户访问的域名
        :return: True=违规需要拦截, False=正常放行
        """
        # 1. 对用户访问的域名进行哈希
        domain_hash = self.hash_domain(domain)
        
        # 2. 与黑名单哈希库比对
        return domain_hash in self.hash_set
```

## 关键点解释

### 1. 盐值（Salt）的作用

```
不加盐：
  "example.com" → SHA256 → "a3f5d8e2..."

加盐后：
  "abc123secret" + "example.com" → SHA256 → "7b9e2d1c..."

优点：
  - 防止黑客用"彩虹表"反向查询
  - 对方公司无法从哈希推算出原始域名
  - 我可以随时更换盐值来"重置"黑名单
```

### 2. 对方公司需要知道的信息

| 信息 | 是否需要提供 | 说明 |
|------|-------------|------|
| 哈希算法（SHA-256） | ✅ 必须 | 双方约定 |
| 盐值（salt） | ✅ 必须 | 双方约定 |
| 哈希库（hashes） | ✅ 必须 | 我提供的黑名单 |
| 明文域名 | ❌ 绝对不能 | 保密 |

### 3. 哈希库的更新流程

```
┌─────────────────────────────────────────────────────────────┐
│                     黑名单更新流程                           │
├─────────────────────────────────────────────────────────────┤

│         │                                                   │
│         ▼                                                   │
│   发现新违规域名                                            │
│         │                                                   │
│         ▼                                                   │
```

## 总结

| 角色 | 操作 | 知道什么 |
|------|------|---------|
| **我（网络安全机构）** | 生成哈希库 | 明文域名 + 盐值 + 哈希算法 |
| **游戏加速器公司** | 哈希比对 | 盐值 + 哈希算法 + 哈希库 |
| **最终效果** | 比对成功 = 拦截 | 无法从哈希反推域名 |

这样设计的**安全性**在于：
- 我提供的只是**哈希值**，对方无法还原出原始域名
- 对方只能检测**我提供的那些域名**，无法自行添加
- 盐值只有我知道，即使哈希库泄露也无法逆向
